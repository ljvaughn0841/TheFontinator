# Datawork

This folder contains font data used in training and code used for processing text to photos.

The following folders containing data are not committed to github:
- fonts-main is a clone of the google fonts repository. (https://github.com/google/fonts/tree/main)
- font-photos is the folder containing the standardized photo's of fonts (extracted by the PhotoExtractor.py)

Note: Adobe Blank is a unique case that is tricky to filter out, so I have deleted it.
